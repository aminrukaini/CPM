How It Works

Forward Pass

Calculates ES (Early Start) and EF (Early Finish) from left to right. ES = max(EF of predecessors), EF = ES + Duration.

Backward Pass

Calculates LF (Late Finish) and LS (Late Start) from right to left. LF = min(LS of successors), LS = LF − Duration.

Critical Path

Activities with Slack = 0 form the critical path — the longest sequence that determines minimum project duration.

Slack / Float

Slack = LS − ES. Non-critical activities can be delayed by their slack without affecting the project deadline.
